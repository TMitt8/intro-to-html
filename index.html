What You'll Learn

  • The basics of HTML
  • How to build the frame of a game
  • Using different tags in your code
  • Different features and functionalities of HTML

The first thing you want to put on your HTML file is <'!DOCTYPE html'>. This will declare the document type and tells the browser how to render the web page. Next you want to add <‘html'>,<‘head'>, and <‘body'> tags underneath. Tags should be closed with <'/element'> for every element We also want to add a title inside the <‘head'> container like this: <‘title'>Tic-Tac-Toe<'/title'> In the future, the head will also include links to our CSS and JS files, but we'll add those when we need them

Your file should look like this so far:

{
  <!DOCTYPE html>
  <html>

  <head>
    <title>Tic-Tac-Toe</title>
  </head>

  <body>

  </body>
  </html>
}

Using Text Tags

HTML offers many different styles of text you can use to organize your webpage and make it look nicer.

  • There are 6 different heading tags, labeled <‘h1'>, <‘h2'>, <‘h3'>, up to <‘h6'>.
  • There's a normal text or paragraph tag labeled <‘p'>
  • There's a tag for hyperlinks labeled <‘a href = "somelink.com'>, where you would put the link after "href"

In this situation, I'll be using an h1 tag and h5 tag, but you can choose which tags you want to use if you feel a different tag looks better.

At the top of the body tag I have the name and some directions:

{
  <h1> Tic Tac Toe!</h1>
  <h5> You will be playing as X. To place an X, click on the tile you want to choose.</h5>
}

Using Ids and Classes

Ids and Classes are incredibly important to add to HTML elements because they help in the future when you want to reference certain elements of your webpage either to improve the style, change the text, add a child, and many other functions. They'll be especially useful when we implement CSS and Javascript in the future.

  • I'm going to add some descriptive ids to my tags:
{
 <h1 id="title"> Tic Tac Toe</h1>
 <h5 id = "directions"> You will be playing as X. To place an X, click on the tile you want to choose.</h5>
}

Using Div

The div tag acts as division between different sections of your webpage. Suppose you only want a certain part of your project to be referenced or styled in a different way than the rest of the body. The div tag allows you to do that. I'm going to add a div tag to store all the rest of our code.

Your code should look similar to this:

{
  <!DOCTYPE html>
  <html>

  <head>
    <title>Tic-Tac-Toe</title>
  </head>

  <body>
    <h1 id="title"> Tic Tac Toe!</h1>
    <h5 id = "directions"> You will be playing as X. To place an X, click on the tile you want to choose.</h5>
    <div id = "enclosed">

    </div>
  </body>
  </html>
}

We're gonna start with the Win-Loss-Tie Counter.

  • First, create another div tag to put this section in
  • Then, use the p tags to create 3 lines of text each representing wins, losses, or ties. Each should start at 0
{
 <p id="wins">Wins: 0</p>
 <p id="losses">Losses: 0</p>
 <p id="ties">Ties: 0</p>
}

Then we're going to use a new tag called button, which creates a button element, to make the reset button. Buttons are useful because they can later be used to execute certain actions when clicked.

{
 <button id="reset">Reset</button>
}

Your code should now look like this:

{
